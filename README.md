# dotfiles

These are my dotfiles, I am aiming at having a minimal setup, with just the things I really need.

I use `oh-my-zsh` for the terminal and `nvim` for editing text.

# setup

First install brew:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

# installation

All the installation process is done via the `install.sh` script.

```bash
cd ~
git clone https://github.com/katistix/dotfiles.git
cd dotfiles
source install.sh
```
